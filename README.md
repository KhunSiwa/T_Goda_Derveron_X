# Travel Booking - Vue 3 + Vite + Tailwind

Project scaffold for a travel booking site using Vue 3 (Composition API), Vue Router, and Tailwind CSS. Contains Home, Search Results, and Hotel Detail pages with mock JSON data and reusable components.

Getting started:

```bash
npm install
npm run dev
```

Files of interest:
- [src/main.js](src/main.js) - app bootstrap
- [src/router/index.js](src/router/index.js) - routes
- [src/store/index.js](src/store/index.js) - simple reactive store
- [src/pages](src/pages) - page components
- [src/components](src/components) - reusable components
